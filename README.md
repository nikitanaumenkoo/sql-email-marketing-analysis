## Email Marketing Analytics (SQL)

### Опис проєкту
SQL-скрипт формує аналітичний датасет для оцінки ефективності email-кампаній
у розрізі дат, країн та характеристик акаунтів.

Проєкт об'єднує дані про акаунти та email-активність користувачів,
дозволяючи аналізувати відправки, відкриття та переходи.

### Джерела даних
- `account` — інформація про акаунти
- `account_session` — зв’язок акаунтів із сесіями
- `session`, `session_params` — параметри сесій (дата, країна)
- `email_sent` — надіслані листи
- `email_open` — відкриття листів
- `email_visit` — переходи з листів

### Реалізована логіка
- об'єднання акаунтних та email-метрик через `UNION ALL`
- повторна агрегація після об'єднання
- розрахунок total-метрик по країнах
- ранжування країн за кількістю акаунтів та email-відправок
- відбір top-10 країн для аналізу

### Ключові метрики
- кількість акаунтів
- надіслані листи
- відкриття email
- переходи з email

### SQL-інструменти
- CTE (WITH)
- JOIN / FULL OUTER JOIN
- UNION ALL
- GROUP BY
- Віконні функції (`DENSE_RANK`)
